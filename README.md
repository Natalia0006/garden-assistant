# Садовый помощник

Локальное веб-приложение для садовода на основе Claude AI.  
Разработано для участка в Краснодарском крае, Абинском районе.

## Возможности

- **Фото-диагностика** — загрузи фото листа, ветки или плода, получи диагноз болезни или вредителя и рекомендации по лечению
- **Сезонный календарь** — список мероприятий по месяцам с учётом климата Абинского района
- **Чат с агрономом** — задай любой вопрос по уходу за своими растениями

## Установка и запуск

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Создай файл `.env`

Скопируй `.env.example` в `.env` и вставь свой API-ключ:

```
ANTHROPIC_API_KEY=sk-ant-ваш-ключ
```

Ключ получить на [console.anthropic.com](https://console.anthropic.com)

### 3. Запусти приложение

```bash
streamlit run app.py
```

Откроется браузер по адресу `http://localhost:8501`

## Растения на участке

| Категория | Культуры |
|---|---|
| Деревья | Персики, абрикосы, айва, яблони, груши, чудо-вишня, миндаль, черешня, инжир, клубничное дерево |
| Кустарники и ягоды | Голубика, смородина, ежевика, малина, крыжовник, клубника, виноград, актинидия, кизил |
| Цветы | Многолетники, однолетники |

## Технологии

- [Streamlit](https://streamlit.io) — интерфейс
- [Anthropic Claude](https://anthropic.com) — AI-модель с поддержкой vision
